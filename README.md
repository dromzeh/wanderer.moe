## Running the Website

In Development:
```bash
  pnpm i && pnpm dev
```

Building:
```bash
  pnpm i && pnpm build
```

Current Cloudflare Pages Configuration:
```
  npx pnpm i --store=node_modules/.pnpm-store && npx pnpm run build
```

You can view `netlify.toml` in the root of the repository to see the old netlify configuration.
